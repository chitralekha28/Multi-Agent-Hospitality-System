# Multi-Agent Hospitality System

An AI-powered Multi-Agent system designed to enhance the hospitality experience by automating customer interactions, recommendations, and service workflows.

---

## 🚀 Project Overview

The Multi-Agent Hospitality System simulates intelligent agents working together to assist users in a restaurant or hospitality environment.

It uses multiple AI agents (Planner, Researcher, Writer, etc.) to:
- Understand user queries
- Retrieve relevant information
- Generate structured and helpful responses

---

## 🧠 Key Features

- 🤖 Multi-Agent Architecture (CrewAI-based)
- 💬 Intelligent Query Handling
- 🌐 Web Search Integration (DuckDuckGo)
- 📝 Structured Response Generation
- ⚡ Modular & Scalable Design
- 🧩 Easy integration with frontend (chatbots / apps)

---

## 🏗️ System Architecture

The system consists of multiple agents:

- **Planner Agent**
  - Breaks down user queries into tasks

- **Researcher Agent**
  - Collects relevant data from web sources

- **Writer Agent**
  - Generates final structured response
